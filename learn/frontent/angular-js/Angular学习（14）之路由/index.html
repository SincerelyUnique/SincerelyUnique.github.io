# 路由的作用

路由就是根据不同 url 地址，动态地让根组件挂载其他组件来实现一个单页面应用。

# 路由的使用

  1. 创建一个带路由的项目，第一个选择选 y
  1. 创建需要的组件
  1. 跟模块引入组件并注入
  2. 路由模块引入组件并在 routes 里面配置 path 和 component
  3. 根页面引入 <router-outlet> 标签
  4. 使用 routerLink 跳转，可以通过 routerLinkActive 设置 active 样式
