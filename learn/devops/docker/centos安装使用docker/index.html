centos 安装使用 docker

首页 学习 DevOPS Docker
  1. 安装 docker，最好 centos 版本 7.x（不是可重装或参考网上 6.5 安装教程），linux 内核 3.1+（不是可升级内核），docker 安装详细可参考菜鸟教程

（1）查看 linux 版本

（2）查看内核版本

（3）安装 docker

（4）关闭 / 启动 docker，也可使用 systemd 操作

  1. 以 spring cloud 项目为例，进入 maven 项目根目录

（1）打包镜像

（2）如果服务器购自类似搬瓦工等国外运营商，无需加速；

（3）如果是国内阿里云等，需做 docker 加速处理，否则超慢，可从 DaoCloud 获取镜像，将镜像链接添加到 /etc/docker/daemon.json 文件中

  1. 查看当前打包好的 docker 镜像，这里我打了两个，jalen/--*
  1. 运行镜像，-p 指后台运行
  1. 页面访问地址，47.98.242.28 是我的阿里云地址，8761 为服务注册中心，8763 为一个服务实例

http://47.98.242.28:8761/

http://47.98.242.28:8763/hi?name=jalen

  1. 查看当前服务器正在运行的 docker 镜像

项目代码参考，可以使用 7z 打成 tar.gz 包到云上再执行以上操作

https://github.com/SincerelyUnique/spring-cloud-demo/tree/master/11docker deploy
