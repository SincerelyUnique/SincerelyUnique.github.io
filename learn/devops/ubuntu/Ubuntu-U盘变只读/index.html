  1. 查看属性
$ sudo mount -o remount.rw /dev/sdb1

mount：根据 mtab，/dev/sdb1 已挂载于 /media/jalen_chu/5426-3BA3
  1. fsck 检查 u 盘数据
$ sudo umount /media/jalen_chu/5426-3BA3
$ sudo fsck /dev/sdb1
fsck，来自 util-Linux 2.20.1
