首页 学习 Go 语言学习 Gin 框架

OS： windows
go： 1.17.2

参考：

  1. https://go.dev/doc/tutorial/web-service-gin
  2. https://github.com/gin-gonic/gin#installation

# 创建文件夹并初始化项目骨架

# 创建 web-service-gin 文件夹

$ mkdir web-service-gin

# 进入 web-service-gin 文件夹

$ cd web-service-gin

# Go mod 初始化

$ go mod init example/web-service-gin

# 获取 gin 依赖

$ go get -u github.com/gin-gonic/gin

# 编码

  1. 在 web-service-gin 文件夹下创建 main.go 文件
  2. 将 main.go 文件首行改为： package main
  3. 编码完成后在 web-service-gin 文件夹下执行： go run .
