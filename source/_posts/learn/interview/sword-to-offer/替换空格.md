# 二维数组中的查找

> 题目描述: 请实现一个函数，将一个字符串中的空格替换成“%20”。例如，当字符串为We Are Happy.则经过替换之后的字符串为We%20Are%20Happy。

## 方法一（投机取巧的方式）

```java
public class Solution {
    public String replaceSpace(StringBuffer str) {
        String s = str.toString();
        s = s.replace(" ","%20");
        return s;
    }
}
```
