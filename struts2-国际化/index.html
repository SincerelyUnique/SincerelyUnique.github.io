  1. struts.xml

<!DOCTYPE struts PUBLIC "-//Apache Software Foundation//DTD Struts Configuration 2.0//EN" "http://struts.apache.org/dtds/struts-2.0.dtd">



<struts>



<constant name="struts.custom.i18n.resources" value="bundles.resourcebundle"/>



<constant name="struts.i18n.encoding" value="UTF-8"/>

<constant name="struts.ui.theme" value="simple" /> 

    <include file="struts-mesLineStatusMonitor.xml"/>

</struts>

2. 和 struts.xml 同级新建 bundles 文件夹，并建 resourcebundle.properties,resourcebundle_en_US.properties,resourcebundle_zh_CN.properties

resourcebundle_en_US.properties

row2.uph							= UPH

resourcebundle_zh_CN.properties

row2.uph							= 每小时产量

如果从上面两个文件找不到，会默认加载 resourcebundle.properties 里面的内容
