上面是一个子组件，在父组件初始化时，即刚进页面时，加载该子组件，该子组件会先执行 create (),

在 create () 方法内有调用 ajax 请求到后端，请求过程中显示 loading，发现此时 v-loading 遮罩默认显示在所有 div 的最上层，

且此时滚动鼠标滑轮，发现遮罩层会向上挪动从而覆盖上部菜单栏，

F12 后发现 v-loading 组件样式属性的 z-index 为 2000，太大，调整为 8 时正好，即需要修改 z-index 属性，

尝试从 css 里写样式覆盖，发现不行，暂时修改如下，但是这样写每个组件都要加 mounted，比较啰嗦，后期有时间在找找其他方法
