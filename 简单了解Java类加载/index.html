简单了解 Java 类加载

发表于 2018-09-30 本文字数 2.6k 字 阅读时长 2 分钟
  • Jalen's Blog
首页

java 中类的加载遵循双亲委派原则，如果你自己写一个 java.lang.String，当我们在启动时，其实加载的是 rt.jar 中的 String 类，不会加载到我们自己写的 String 类，即你永远不会改变 jdk 自带的 String 类
